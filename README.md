# Backup EC2

---

## Prerequisites

1. Python 3
2. pip3

```shell
    $ sudo apt install python3-pip
```

3. boto3

```shell
    $ pip3 install boto3
```

---

## Getting Started

1. Modify .env

```shell
    $ cp env.example .env
```

2. fill in .env

3. Run program

```shell
    $ python3 index.py
```
